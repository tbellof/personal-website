# thiagobello.com (personal website)

Minimal personal site — React + TypeScript + Vite. Inspired by [zeviarnovitz.com](https://zeviarnovitz.com/).

**Planning, copy, and commands** live in the separate workspace repo:

`Build AI Product Sense/Projects/personal-website/`

## Quick start

```bash
npm install
npm run dev
```

```bash
npm run build   # production bundle → dist/
npm run preview # preview production build
```

## Deploy

Connect this repo to [Vercel](https://vercel.com) (Vite preset). Add custom domain when ready.

See planning repo: `Projects/personal-website/SETUP-github.md` for first-time GitHub setup.
